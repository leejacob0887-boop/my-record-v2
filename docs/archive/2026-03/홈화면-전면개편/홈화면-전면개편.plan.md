## 1. 목표

현재 홈 화면(카드 그리드 구조)을 **카카오톡 스타일 AI 채팅 UI**로 완전 교체.

앱을 열자마자 AI와 대화하듯 기록을 시작할 수 있는 경험 제공.

---

## 2. 요구사항

### 2.1 화면 구조 (5개 영역)

```
┌─────────────────────────────┐
│  상단바                      │ ← 달 아이콘 / Notia 로고 / 설정 아이콘
├─────────────────────────────┤
│  카테고리 가로 스크롤          │ ← 일기/메모/아이디어/캘린더/투두
├─────────────────────────────┤
│  ─────── 구분선 ──────────   │
├─────────────────────────────┤
│  AI 채팅 영역 (스크롤)        │ ← 카카오톡 스타일 말풍선
│                             │
│  [AI] 안녕하세요...  오전9:00  │
│         오늘 할일 뭐야? [User] │
│  [AI] 오늘 할일 3개...        │
├─────────────────────────────┤
│  하단 입력창 영역 (고정)       │
│  [오늘일정][오늘할일][최근기록][날씨] │
│  [+][🎤][말하거나 입력하세요...][📷][▶] │
└─────────────────────────────┘
```

### 2.2 영역별 상세 스펙

#### A. 상단바
- 좌: 달 아이콘 (다크모드 토글, 현재 DarkModeToggle 컴포넌트 유지)
- 중: Notia 로고 (깃털펜 SVG 아이콘 + "Notia" 텍스트)
- 우: 설정 아이콘 → `/settings` 링크

#### B. 카테고리 가로 스크롤
레퍼런스 이미지 기준 5개 아이콘:

| 항목 | 아이콘 | 색상 | 링크 |
|------|--------|------|------|
| 일기 | BookOpen | `#0F6E56` (진한 Teal) | `/diary` |
| 메모 | Zap | `#1D9E75` (중간 Teal) | `/moments` |
| 아이디어 | Lightbulb | `#E1F5EE` bg + `#0F6E56` 아이콘 | `/ideas` |
| 캘린더 | CalendarDays | `#9FE1CB` (연한 Teal) | `/calendar` |
| 투두 | CheckSquare | `#085041` (어두운 Teal) | `/todo` |

- 각 아이콘: 64×64 rounded-2xl 배경 + 아이콘 + 라벨 텍스트
- 가로 스크롤 (overflow-x-auto, scrollbar 숨김)

#### C. 구분선
- `<hr>` 또는 `border-t` 1px gray-200

#### D. AI 채팅 영역
카카오톡 스타일:

**AI 말풍선 (왼쪽)**
```
[🪶] [흰색 말풍선 내용]
              오전 9:00
```
- 왼쪽: Teal(#0F6E56) 원형 아이콘 (깃털펜 아이콘)
- 중: 흰색 bg, rounded-2xl (좌상단 직각), 그림자 subtle
- 우하: 시간 텍스트 (text-xs text-gray-400)

**사용자 말풍선 (오른쪽)**
```
오후 2:30
       [Teal 말풍선 내용]
```
- 우: `#0F6E56` bg 말풍선, 흰색 텍스트
- 좌하: 시간 텍스트

**초기 AI 메시지**
앱 첫 진입 시 AI가 인사 메시지 표시:
> "안녕하세요! 😊 오늘 어떤 걸 기록할까요?"

#### E. 하단 입력창 영역 (고정, 흰색 배경)

**빠른 버튼 행** (Chip 스타일):
- `오늘 일정` / `오늘 할일` / `최근 기록` / `오늘 날씨`
- 클릭 시 → 각 바텀시트 열림

**입력창 행**:
```
[+] [🎤] [말하거나 입력하세요...          ] [📷] [▶]
```
- `+`: 추가 옵션 (미사용 시 플레이스홀더)
- `🎤`: 마이크 (기존 음성인식 로직 연결)
- 텍스트 입력: placeholder "말하거나 입력하세요..."
- `📷`: 카메라 (기존 카메라 로직 연결)
- `▶`: 전송 버튼 (Teal)

### 2.3 빠른 버튼 → 바텀시트

| 버튼 | 바텀시트 내용 |
|------|--------------|
| 오늘 일정 | 시간별 일정 목록 (캘린더 events) |
| 오늘 할일 | 체크박스 체크 가능한 투두 목록 |
| 최근 기록 | 태그+내용+시간 목록 (기존 recentItems 로직) |
| 오늘 날씨 | 날씨 카드 (OpenWeatherMap API 연동) |

### 2.4 AI 채팅 로직
- 사용자가 텍스트 입력 후 전송 → 기존 `/api/chat` POST 호출 (mode: 'save')
- AI 응답 → 채팅 영역에 말풍선으로 표시
- 마이크 → 음성인식 후 텍스트로 변환하여 채팅에 추가 + 저장
- 저장 성공 시 AI가 확인 메시지 응답

## 3. 기술 결정

### 3.1 수정 범위
- **교체**: `src/app/page.tsx` — 완전 새 코드로 교체
- **유지**: 기존 hooks (useDiary, useMoments, useIdeas, useAuth), uploadImage, `/api/chat`
- **추가**: OpenWeatherMap API 연동 (`/api/weather` 또는 클라이언트 직접 호출)
- **건드리지 않음**: 다른 페이지 일체

### 3.2 날씨 API
- OpenWeatherMap Free Tier 사용
- 환경변수: `NEXT_PUBLIC_OPENWEATHER_API_KEY`
- 위치: 브라우저 Geolocation API → 위도/경도 → 날씨 조회

### 3.3 채팅 상태 관리
- `useState<Message[]>` — 세션 중 채팅 히스토리 유지
- 페이지 새로고침 시 초기화 (localStorage 저장 옵션은 추후)

### 3.4 컴포넌트 구조 (신규)
```
page.tsx (홈)
├── HomeTopBar          — 상단바
├── CategoryScroll      — 카테고리 가로 스크롤
├── ChatArea            — AI 채팅 영역
│   ├── AIBubble        — AI 말풍선
│   └── UserBubble      — 사용자 말풍선
├── BottomInputArea     — 하단 입력창 고정
│   ├── QuickButtons    — 빠른 버튼 행
│   └── ChatInput       — 입력창 행
└── BottomSheet         — 공통 바텀시트 래퍼
    ├── TodayScheduleSheet
    ├── TodayTodoSheet
    ├── RecentRecordsSheet
    └── WeatherSheet
```

## 4. 구현 우선순위

1. 기본 레이아웃 (상단바 + 카테고리 + 구분선 + 채팅영역 + 하단입력)
2. AI 채팅 말풍선 + 텍스트 전송 기능
3. 마이크/카메라 기존 로직 연결
4. 빠른버튼 + 바텀시트 (최근기록 먼저, 투두, 일정 순)
5. 날씨 바텀시트 (OpenWeatherMap 연동)

---

## 5. 제약사항

- 다른 페이지 (diary, moments, ideas, calendar, todo, settings 등) 건드리지 않음
- 브랜드 컬러 #0F6E56 일관 적용
- 모바일 최우선 (100dvh, max-w-[430px])
- 기존 useDiary/useMoments/useIdeas hooks 그대로 사용
