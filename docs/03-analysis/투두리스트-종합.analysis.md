## Plan 없이 구현된 기능 (5개)

### 1. 스와이프 완료/삭제 (TodoItem.tsx)
- 오른쪽 스와이프 > 72px → 완료 토글 (녹색 배경)
- 왼쪽 스와이프 > 72px → 삭제 (빨간 배경)
- directionRef로 수평/수직 판별 → 드래그와 스와이프 충돌 없음

### 2. 완료 항목 접기/펼치기 + 전체삭제 (TodoList.tsx)
- doneExpanded state로 접기/펼치기
- "완료 N개" 카운트 표시
- 전체삭제: 2단계 인라인 확인 UX

### 3. 반복 할일 (todos.ts, TodoInput.tsx, TodoItem.tsx, page.tsx)
- 타입: daily | weekly | monthly
- UI: 반복 없음/매일/매주/매월 선택 버튼
- 완료 시 getNextDueDate()로 다음 날짜 계산 후 자동 생성
- RefreshCw 뱃지 + "매일/매주/매월" 텍스트 표시

### 4. 완료 토스트 (TodoToast.tsx + page.tsx)
- 9개 랜덤 격려 메시지, 2.8초 표시
- animate-fade-in-up 애니메이션 (globals.css)

### 5. 탭 순서 변경
- 기본 탭: 전체(all) / 오른쪽: 오늘
