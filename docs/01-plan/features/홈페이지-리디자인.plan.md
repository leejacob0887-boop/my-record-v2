# 홈페이지-리디자인 Planning Document

> **Summary**: 단순 카드 나열에서 벗어나, 오늘 날짜·최근 기록 미리보기·빠른 기록 버튼을 갖춘 풍부한 홈 화면으로 개선
>
> **Project**: my-record-v2
> **Version**: 0.2.0
> **Author**: jacob
> **Date**: 2026-03-09
> **Status**: Draft

---

## Executive Summary

| Perspective | Content |
|-------------|---------|
| **Problem** | 현재 홈은 4개 카드만 나열되어 있어, 오늘 뭘 기록했는지·최근 기록이 무엇인지 한눈에 알 수 없음 |
| **Solution** | 오늘 날짜 헤더, 빠른 기록 버튼, 최근 기록 미리보기 섹션을 추가해 홈 화면을 대시보드화 |
| **Function/UX Effect** | 앱 진입 즉시 오늘의 상태 파악 + 원탭 기록 시작 → 사용 빈도 증가 기대 |
| **Core Value** | 기록 앱의 홈은 "지금 바로 쓸 수 있는 공간"이어야 한다 |

## 1. Overview

### 1.1 현재 상태 (AS-IS)

```
┌─────────────────────────┐
│  나의 기록               │
│  오늘의 생각을 기록...    │
│                         │
│  [일기]    [지금이순간]  │
│  [아이디어] [캘린더]     │
└─────────────────────────┘
```

**문제점**:
- 오늘 날짜/요일 정보 없음
- 기록 수만 표시, 내용 미리보기 없음
- 새 기록을 시작하려면 카드 → 목록 → 버튼 (2단계)
- 캘린더 카드에 count=0 하드코딩

### 1.2 목표 상태 (TO-BE)

```
┌─────────────────────────┐
│  안녕하세요 👋           │
│  2026년 3월 9일 월요일   │  ← 오늘 날짜
│                         │
│  ┌─────────────────────┐│
│  │ ⚡ 빠른 기록         ││  ← Quick Action Bar
│  │ [일기] [순간] [아이디어]││
│  └─────────────────────┘│
│                         │
│  오늘의 기록             │  ← 오늘 기록 섹션
│  (없으면: "아직 없어요") │
│                         │
│  최근 기록               │  ← 최신 3개 미리보기
│  ...                    │
│                         │
│  [일기]    [지금이순간]  │  ← 기존 카드 (하단)
│  [아이디어] [캘린더]     │
└─────────────────────────┘
```

## 2. Scope

### 2.1 In Scope

- [ ] **헤더**: 인사말 + 오늘 날짜·요일 표시
- [ ] **Quick Action Bar**: 일기/순간/아이디어 빠른 작성 버튼 (각 `/diary/new`, `/moments/new`, `/ideas/new` 이동)
- [ ] **오늘의 기록 섹션**: 오늘 날짜의 일기·순간·아이디어 요약 표시
- [ ] **최근 기록 미리보기**: 전체 유형 통합, 최신 3개 카드
- [ ] **기존 유형 카드 유지**: 하단에 4개 카드 (캘린더 count 제거)
- [ ] 모바일 최적화 유지 (max-w-md, Tailwind CSS)

### 2.2 Out of Scope

- 다크모드
- 사용자 이름 설정 (하드코딩 "안녕하세요")
- 통계/차트
- 검색 기능

## 3. Requirements

### 3.1 Functional Requirements

| ID | Requirement | Priority |
|----|-------------|----------|
| FR-01 | 오늘 날짜를 한국어 형식으로 표시 (예: 2026년 3월 9일 월요일) | High |
| FR-02 | Quick Action Bar: 일기/순간/아이디어 3개 버튼, 각 new 페이지로 이동 | High |
| FR-03 | 오늘의 기록: 일기(있으면 제목), 순간(개수), 아이디어(개수) | High |
| FR-04 | 최근 기록 미리보기: 3가지 유형 통합, 최신 3개, 클릭 시 상세 이동 | Medium |
| FR-05 | 캘린더 카드의 count=0 하드코딩 제거 | Low |
| FR-06 | 오늘 기록이 없으면 "아직 기록이 없어요" 빈 상태 메시지 | Medium |

### 3.2 Non-Functional Requirements

| Category | Criteria |
|----------|----------|
| 모바일 | 375px 이상 정상 동작 |
| 성능 | LocalStorage 읽기 최소화 (기존 훅 재사용) |
| 일관성 | 기존 Tailwind 스타일 (rounded-2xl, border-gray-100, shadow-sm) 유지 |
