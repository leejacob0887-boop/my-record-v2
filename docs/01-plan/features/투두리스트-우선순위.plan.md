# 투두리스트 우선순위 Plan

## 1. 개요

기존 투두리스트에 우선순위(priority) 필드 추가. 높음/중간/낮음 3단계를 색상으로 구분하고 높은 순으로 정렬.

## 2. 요구사항

| ID | 기능 | 우선순위 |
|----|------|---------|
| F-01 | 투두 추가 시 우선순위 선택 (높음/중간/낮음) | P0 |
| F-02 | 높음 → 빨간색, 중간 → 주황색, 낮음 → 회색 표시 | P0 |
| F-03 | 목록을 우선순위 높은 순으로 정렬 | P0 |
| F-04 | 기본값: 중간 | P0 |

## 3. 기술 결정

| 항목 | 결정 |
|------|------|
| DB | todos 테이블에 `priority` 컬럼 추가 (Supabase ALTER TABLE) |
| 타입 | `'high' \| 'medium' \| 'low'` |
| 정렬 | DB 쿼리 order by priority (high→medium→low) 또는 클라이언트 정렬 |
| 기본값 | `'medium'` |

## 4. 파일 변경 목록

| 파일 | 변경 | 설명 |

## 5. Supabase SQL

```sql
alter table todos add column priority text not null default 'medium'
  check (priority in ('high', 'medium', 'low'));
```
