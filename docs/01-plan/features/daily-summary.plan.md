## 1. 기능 개요

### 1.1 요약하기 버튼

캘린더 페이지 "오늘의 기록" 카드 하단에 "✨ 오늘 하루 요약하기" 버튼 배치.
오늘 기록이 하나도 없으면 버튼 비활성화.

### 1.2 AI 요약 생성

버튼 클릭 → 이미 로드된 훅 데이터(diaryEntries, moments, ideas)에서 오늘 기록 수집 → `POST /api/daily-summary` 호출 → Anthropic Haiku로 감성 요약 생성.

**데이터 수집 전략**: 별도 Supabase fetch 없이 캘린더 페이지에 이미 마운트된 훅 상태 사용.

### 1.3 팝업 모달

요약 완료 시 배경 딤 + 카드 팝업 (slide-up 애니메이션).
팝업 내: 날짜, 요약 텍스트, "메모로 저장" + "닫기" 버튼.

### 1.4 메모로 저장

"메모로 저장" 클릭 → `useMoments().add()` 호출 → 저장 완료 후 SavePreviewCard로 완료 피드백.

## 3. 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| AI | Anthropic `claude-haiku-4-5-20251001` | 기존 chat API와 동일 모델 |

| 팝업 | 인라인 JSX + `animate-slide-up-toast` | 외부 라이브러리 없음 |


### 4.3 API 설계

**Request**:
```json
{
  "date": "2026-03-12",
  "diary": [{ "title": "...", "content": "..." }],
  "moments": [{ "text": "..." }],
  "ideas": [{ "title": "...", "content": "..." }]
}
```

**Response**:
```json
{ "summary": "오늘 하루도 참 바쁘고 풍성했네요. ..." }
```

**시스템 프롬프트 핵심**:
```
당신은 사용자의 하루 기록을 읽고 따뜻하게 요약해주는 AI입니다.
일기·메모·아이디어를 종합해 2~4문장으로 감성적으로 요약하세요.
사용자를 격려하고 하루를 마무리하는 따뜻한 톤으로 작성하세요.
```
